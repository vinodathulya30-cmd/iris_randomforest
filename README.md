                              Iris Classification using Random Forest & Streamlit

A machine learning web application that classifies Iris flower species using the Random Forest Classification algorithm. The frontend is built with Streamlit for an interactive and user-friendly experience.

Features
Predicts Iris flower species:
Setosa
Versicolor
Virginica
Uses Random Forest Classification for accurate predictions
Interactive web interface with Streamlit
Simple and beginner-friendly project structure
Real-time predictions based on user input
Technologies Used
Python
Scikit-learn
Pandas
NumPy
Streamlit
Random Forest Classifier
Machine Learning Algorithm

This project uses the Random Forest Classification algorithm, an ensemble learning method that combines multiple decision trees to improve prediction accuracy and reduce overfitting.

Dataset

The model is trained on the famous Iris dataset, which contains:

Sepal Length
Sepal Width
Petal Length
Petal Width
